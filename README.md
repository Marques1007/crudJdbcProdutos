# CRUD de produtos - Java + JDBC
Projeto para cadastro, edição, exclusão e consulta de dados de produtos
implementado em Java com JDBC e acessando um banco de dados PostgreSQL.

---

## Tecnologias e padrões utilizados:
 - Java 21
 - JDBC
 - PostgreSQL
 - Arquitetura em camadas (controllers, entities, factories e repositories)
